# Project Overview - Jetschool Academy

A funnel website with 4 program types (Webinar, Kelas Online (LMS), Workshop, Bootcamp), Xendit payment integration, automated WhatsApp messages (Evolution API), post-tests, e-certificates with QR, and an admin panel.

## Tech Stack
- Framework: Next.js 16 (App Router) + React 19 + TypeScript
- Database & ORM: Prisma 6 + MySQL (InnoDB)
- Design: Bento card aesthetics (colors: purple, orange, lime, Plus Jakarta Sans font) - minimal text, repeat CTAs, focus on conversion.
